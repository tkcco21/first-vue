# First Vue (ver. TODO)

developブランチに切り替え
```
git checkout -b develop origin/develop
```
package.jsonに記載のあるパッケージのインストール
```
yarn
```

インストールしたら下記2つを別タブで実行

```
yarn dev:server
```

```
yarn dev:watch
```
